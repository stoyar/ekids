Полезные теги для домашки

<body> - содержит всё, что есть на странице

<header> - заглавная часть страницы

<main> - основное содержимое

<footer> - "подвал", содержит контакты, ссылки, копирайт

<nav> - навигация. Используют для меню навигации по сайту

<h1> - <h6> - заголовки 1 - 6 уровня

<p> - (параграф) - текст, просто текст

<div> - блочный контейнер, "коробочка" для других тегов

<ul> - маркированный список

<ol> - нумерованный список

<li> - элемент списка

<img src="" alt=""> - картинка

<span> - оборачивает фрагмент текста, без переноса строки

<br> - перенос строки

<hr> - горизонтальная линия

<a href=""> - ссылка
