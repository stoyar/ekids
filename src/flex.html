Флексбокс - это сила! :)

Стили для родительского элемента

display: flex - делает элемент флекс контейнером

flex-direction - направление выравнивания потомков

значения: row column

flex-wrap - перенос потомков на новую строку

значения: wrap nowrap

justify-content - выравнивание потомков вдоль главной оси

значения: flex-start flex-end center space-between space-around space-evenly

align-items - выравнивание потомков вдоль второстепенной оси

значения: flex-start flex-end center baseline stretch

Стили для потомков

align-self - выравнивание этого потомка вдоль второстепенной оси
